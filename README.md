# Projet Harmony Network
Harmony Network est un petit réseau social simpliste, créé dans le cadre d'un projet `php` à l'IUT de Nice.

## Fonctionnement
Ce réseau social utilise différents langages :
- PHP
- HTML 5
- CSS
- JAVASCRIPT
